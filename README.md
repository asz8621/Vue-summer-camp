# Vue-summer-camp
## Week 1 - Vue 3 起步走

透過資料內容呈現多筆資料列表
- 資料來源：https://randomuser.me/
- 需要使用 v-for 呈現多筆資料
- 需要呈現至少 5 個物件內的屬性值
- 文字、圖片、連結等
- 嘗試使用 v-on 來進行資料的切換
